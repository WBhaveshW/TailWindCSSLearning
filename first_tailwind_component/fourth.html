What does the -rotate-180 class do in Tailwind CSS?

The -rotate-180 class in Tailwind CSS applies a CSS transform that rotates an element 180 degrees counterclockwise. This is often used to flip icons or images upside down.

What does the cursor-pointer class do in Tailwind CSS?

The cursor-pointer class in Tailwind CSS changes the mouse cursor to a pointer (hand icon) when hovering over an element. This indicates that the element is clickable, such as a button or link.

What does the hidden class do in Tailwind CSS?

The hidden class in Tailwind CSS is used to completely hide an element from the page. When applied, the element will not take up any space in the layout, and it will not be visible to users.

What does the justify-between class do in Tailwind CSS?

The justify-between class is used in a flex container to distribute space between child elements. It places the first item at the start and the last item at the end, with equal space distributed between all items in between.

What is the purpose of the aria-controls attribute in HTML?

The aria-controls attribute is used to indicate that an element controls the visibility or state of another element. This is particularly useful for accessibility, allowing assistive technologies to understand relationships between elements.
Usage: The value of aria-controls should be the id of the element that is being controlled. For example, it can be used in a button that toggles a dropdown or a modal.

What does the items-center class do in Tailwind CSS?

The items-center class is used in a flex or grid container to vertically align child elements in the center of the container along the cross axis. This is particularly useful for centering items within a row or column layout.

What does the transition-transform class do in Tailwind CSS?

The transition-transform class in Tailwind CSS applies a smooth transition effect to the transform property of an element. This means that any changes to transformation properties (like scale, rotate, translate) will animate over a specified duration instead of changing instantly.

What are the <dt> and <dd> elements in HTML?

The <dt> and <dd> elements are used to create description lists. A description list is a type of list where you can pair terms with descriptions, commonly used for glossaries or to provide details about various items.

What does toggling a class in JavaScript do?

Toggling a class in JavaScript means adding or removing a CSS class from an element's class list. This is commonly done to change the styling or behavior of an element when an event occurs, such as a click.
Using classList.toggle()
  • If the specified class exists on the element, toggle() removes it.
  • If the class does not exist, toggle() adds it.
