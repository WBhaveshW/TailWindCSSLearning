Tailwind CSS Classes

Class-1 : scale-105

Scales the element to 105% of its original size.

border-radius: 0.25rem; /* 4px by default */

Class-2 : rounded

Rounds the corners.

transition: all 0.3s ease-in-out; /* Duration and timing function can be specified here */

Class-3 : transition

Enables transitions on all properties.

/* No specific property needed; it's used with transform functions like scale */

Class-4: transform

Enables transformation for scaling.

transition-duration: 300ms;

Class-5 : duration-300

Sets the duration of the transition to 300ms.

transition-timing-function: ease-in-out;

Class-6 : ease-in-out

Sets the timing function to ease in and out.

&:hover { transform: scale(1.05); }

Class-7 : hover:scale-105

Scales the button to 105% on hover.

box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06);

Class-8 : shadow-md

Applies a medium box shadow, giving the card depth.

.container { display: grid; grid-template-columns: repeat(1, 1fr); /* Default: 1 column */ gap: 24px; /* 1.5rem */ margin-left: 12px; /* 0.75rem */ margin-right: 12px; /* 0.75rem */ }

Class-9 : grid-cols-1

This sets the grid to have 1 column by default. On small screens, it will display one item per row.

@media (min-width: 640px) { .container { grid-template-columns: repeat(2, 1fr); /* 2 columns on small screens */ } }

Class-10 : sm:grid-cols-2

This class changes the grid to 2 columns when the screen size reaches the small breakpoint (typically 640px and above). At this size, there will be two items per row.

@media (min-width: 1024px) { .container { grid-template-columns: repeat(3, 1fr); /* 3 columns on large screens */ } }

Class-11 : lg:grid-cols-3

This sets the grid to 3 columns at the large breakpoint (typically 1024px and above). Therefore, three items will be displayed per row on larger screens.

.container { display: grid; grid-template-columns: repeat(1, 1fr); /* Default: 1 column */ gap: 24px; /* 1.5rem */ margin-left: 12px; /* 0.75rem */ margin-right: 12px; /* 0.75rem */ }

Class-12 : gap-6

This adds a gap of 1.5rem (24 pixels) between the grid items, creating space between the rows and columns.

.element { margin-left: 12px; /* or 0.75rem */ margin-right: 12px; /* or 0.75rem */ }

Class-12 : mx-3

This applies horizontal margins of 0.75rem (12 pixels) on both the left and right sides of the grid container, ensuring some space from the edges of the viewport.
